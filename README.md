# lwcs

本项目是一个本地化的游戏测试与控制工具，采用 **Python Flask 后端 + Web 前端** 的组合，提供统一的页面操作入口与接口服务，便于进行账号流程、角色流程、战斗流程与数据观测相关的联调测试。

## 主要功能

- 本地 Web 控制台：通过浏览器统一操作测试流程。
- 连接与会话管理：提供登录、选角、断开连接等基础流程能力。
- 战斗流程控制：支持战斗触发、循环控制与相关配置管理。
- 物品与背包操作：支持背包查看、物品使用/整理等常用测试动作。
- 实时状态推送：通过 SSE 向前端推送状态与事件变化。
- 测试数据管理：支持常用配置项的持久化与读取（如快捷登录、常用条目等）。
- 报文观测能力：提供报文记录查询、标注与辅助解析接口（用于测试定位）。

## 项目结构（概览）

```text
lwcs/
├─ game_test/
│  ├─ api/          # HTTP API 与 SSE 推送入口
│  ├─ core/         # 基础能力
│  ├─ features/     # 业务功能
│  ├─ services/     # 服务层
│  ├─ web/          # 前端页面
│  └─ run.py        # 应用启动入口
├─ start_lwcs.bat   # Windows 启动脚本
```

## 运行方式（Windows）

1. 准备 Conda 环境：`conda activate lwcs`
2. 安装依赖：`pip install -r game_test/requirements.txt`
3. 启动项目：`python -m game_test.run`
4. 浏览器访问：`http://127.0.0.1:7896/`

也可直接使用 `start_lwcs.bat` 启动（脚本内已包含 Conda 环境激活命令）。
